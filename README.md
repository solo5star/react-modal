# react-modal

react-modal은 React를 위한 가장 간단한 모달 구현을 제공해주는 라이브러리입니다.

## Features

* ESM과 CJS 모두 지원합니다.

## Getting Started

이 라이브러리를 사용하기 위해선 `react@^18.0.0`, `react-dom@^18.0.0`, `styled-components@^6.0.0` 이 필요합니다.

```bash
npm install react react-dom styled-components
```

For npm:
```bash
npm install react-modal
```
